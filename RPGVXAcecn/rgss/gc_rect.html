Rect

矩形的类。

超类

  • Object

类方法

Rect.new(x, y, width, height)
Rect.new (RGSS3)

创建一个 Rect 对象。

未指定参数时，默认值为 (0, 0, 0, 0)。(RGSS3)

方法

set(x, y, width, height)
set(rect) (RGSS3)

一次设置所有属性。

第二种形式会从另一个 Rect 对象上复制所有的属性。(RGSS3)

empty

将所有属性设置为 0。

属性

x

矩形左上角的 X 坐标。

y

矩形左上角的 Y 坐标。

width

矩形的宽度。

height

矩形的高度。
