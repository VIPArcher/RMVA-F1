RPG::Event::Page

地图事件块事件页资料。

超类

  • Object

调用者

  • RPG::Event

属性

condition

事件指令的执行条件（RPG::Event::Page::Condition）。

graphic

事件页指定事件块的图片（RPG::Event::Page::Graphic）。

move_type

移动类型（0：固定、1：随机、2：接近、3：自定义）。

move_speed

移动速度（1：1/8 倍速、2：1/4 倍速、3：1/2 倍速、4：正常、5：2 倍速、6：4 倍速）。

move_frequency

移动频率：（1：最低、2：低、3：正常、4：高、5：最高）。

move_route

移动路线（RPG::MoveRoute）。只有当移动类型为「自定义」时才会调用。

walk_anime

步行动画的开关。

step_anime

踏步动画的开关。

direction_fix

方向锁定的开关。

through

允许穿透的开关。

priority_type

优先级（0：在普通角色下方、1：与普通角色同层、2：在普通角色上方）。

trigger

触发条件（0：按确定键、1：玩家接触、2：事件接触、3：自动执行、4：并行处理）。

list

执行内容。RPG::EventCommand 的数组。

内部类

  • RPG::Event::Page::Condition
  • RPG::Event::Page::Graphic

定义

class RPG::Event::Page
  def initialize
    @condition = RPG::Event::Page::Condition.new
    @graphic = RPG::Event::Page::Graphic.new
    @move_type = 0
    @move_speed = 3
    @move_frequency = 3
    @move_route = RPG::MoveRoute.new
    @walk_anime = true
    @step_anime = false
    @direction_fix = false
    @through = false
    @priority_type = 0
    @trigger = 0
    @list = [RPG::EventCommand.new]
  end
  attr_accessor :condition
  attr_accessor :graphic
  attr_accessor :move_type
  attr_accessor :move_speed
  attr_accessor :move_frequency
  attr_accessor :move_route
  attr_accessor :walk_anime
  attr_accessor :step_anime
  attr_accessor :direction_fix
  attr_accessor :through
  attr_accessor :priority_type
  attr_accessor :trigger
  attr_accessor :list
end
