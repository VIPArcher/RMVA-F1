RPG::BaseItem

角色、职业、技能、物品、武器、护甲、敌人和状态的超类。

有些数据并不需要所有的属性，但为了方便，它们也包含了进来。

超类

  • Object

属性

id

ID。

name

名称。

icon_index

图标编号。

description

说明。

features

特性列表。RPG::BaseItem::Feature 的数组。

note

备注。
