RPG::Enemy::DropItem

敌人掉落物品的数据类。

超类

  • Object

调用者

  • RPG::Enemy

属性

kind

掉落物品的类型。

  • 0: 无
  • 1: 物品
  • 2: 武器
  • 3: 护甲
data_id

掉落物在对应数据库中的编号。

denominator

掉率的分母。

定义

class RPG::Enemy::DropItem
  def initialize
    @kind = 0
    @data_id = 1
    @denominator = 1
  end
  attr_accessor :kind
  attr_accessor :data_id
  attr_accessor :denominator
end
