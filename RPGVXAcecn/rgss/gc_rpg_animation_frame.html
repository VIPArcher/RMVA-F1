RPG::Animation::Frame

动画帧的数据类。

超类

  • Object

调用者

  • RPG::Animation

属性

cell_max

单元个数。等于帧中单元的最大编号。

cell_data

包含单元内容的二维数组（Table）。

使用 cell_data[cell_index, data_index] 的形式。

data_index 的范围是从 0 到 7 ，代表单元的各项信息。（0：图案，1：X 坐标，2：Y 坐标，3：缩放度，4：旋转角度，5：左右翻转，6：不透明度，7：合成方式。图案是编辑器中的显示的编号减 1。-1 表示不存在该单元。

定义

class RPG::Animation::Frame
  def initialize
    @cell_max = 0
    @cell_data = Table.new(0, 0)
  end
  attr_accessor :cell_max
  attr_accessor :cell_data
end
