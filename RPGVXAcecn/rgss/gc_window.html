Window

游戏窗口的类，内部由多个精灵构成。

超类

  • Object

类方法

Window.new([x, y, width, height]) (RGSS3)

创建一个窗口对象，根据需要指定位置与大小。

方法

dispose

释放窗口。若窗口已释放则什么都不做。

disposed?

当窗口已释放则返回 true。

update

刷新光标的闪烁和暂停标记的动画。原则上每帧调用一次。

move(x, y, width, height) (RGSS3)

同时设置窗口的 X 坐标、Y 坐标、宽度、高度。

open? (RGSS3)

如果窗口完全打开（openness == 255）则返回 true。

close? (RGSS3)

如果窗口完全关闭（openness == 0）则返回 true。

属性

windowskin

对窗口皮肤位图（Bitmap）的引用。

皮肤规格和 RMVX 基本相同，详情请参见 素材规格：窗口皮肤。

contents

对窗口内容位图（Bitmap）的引用。

cursor_rect

光标矩形（Rect）。

以窗口内容为基准指定坐标。(RGSS3)

viewport

对窗口关联的显示端口（Viewport）的引用。

active

光标闪烁的状态。true 表示闪烁。初始值为 true。

visible

窗口可见的状态。true 表示可见。初始值为 true。

arrows_visible (RGSS3)

滚动箭头可见的状态，true 表示可见。初始值为 true。

pause

暂停标记可见的状态。暂停标记是表示消息窗口等待输入的记号。true 表示可见。初始值为 false。

x

窗口的 X 坐标。

y

窗口的 Y 坐标。

width

窗口的宽度。

height

窗口的高度。

z

窗口的 Z 坐标。数值愈大的平面愈靠近玩家。

Z 坐标相同的，最后生成的对象则会最靠近玩家。

初始值为 100。 (RGSS3)

ox

窗口内容原点的 X 坐标。修改此数值可以滚动窗口。

该属性同样作用于光标。 (RGSS3)

oy

窗口内容原点的 Y 坐标。修改此数值可以滚动窗口。

该属性同样作用于光标。 (RGSS3)

padding (RGSS3)

窗口框架与内容之间的边距，初始值为 12。 (RGSS3)

padding_bottom (RGSS3)

下方专用的 padding。由于该属性会随着 padding 的改变而改变，因此必须在 padding 之后设置。

opacity

窗口的不透明度（0～255）。超出此范围的数值会自动修正。初始值为 255。

back_opacity

窗口背景的不透明度（0～255）。超出此范围的数值会自动修正。初始值为 192。(RGSS3)

contents_opacity

窗口内容的不透明度（0～255）。超出此范围的数值会自动修正。初始值为 255。

openness

窗口的打开程度（0～255）。超出此范围的数值会自动修正。

将此数值从 0（完全关闭）至 255（完全打开）之间变换，可以产生窗口打开和关闭的动画效果。openness 不满 255 时，窗口的内容不会显示。初始值为 255。

tone (RGSS3)

窗口背景的色调（Tone）。
