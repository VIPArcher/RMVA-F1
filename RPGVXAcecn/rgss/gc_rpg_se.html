RPG::SE

SE 的数据类。拥有通过 Audio 模块播放自身的功能。

超类

  • RPG::AudioFile

调用者

  • RPG::Animation::Timing
  • RPG::System
  • RPG::EventCommand
  • RPG::MoveCommand

类方法

RPG::SE.stop

停止播放 SE。

方法

play

开始播放 SE。

定义

class RPG::SE < RPG::AudioFile
  def play
    unless @name.empty?
      Audio.se_play('Audio/SE/' + @name, @volume, @pitch)
    end
  end
  def self.stop
    Audio.se_stop
  end
end
