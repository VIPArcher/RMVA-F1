RPG::Enemy::Action

敌人[行为模式]的数据类。

超类

  • Object

调用者

  • RPG::Enemy

属性

skill_id

技能 ID。

condition_type

行动条件。

  • 0: 平时
  • 1: 回合数
  • 2: 体力值
  • 3: 魔力值
  • 4: 状态
  • 5: 队伍等级
  • 6: 开关
condition_param1
condition_param2

条件的参数，所有的类型共用。

举例来说，当条件为[体力值]时，condition_param1 是下限而 condition_param2 是上限。

rating

优先级（1..10）。

定义

class RPG::Enemy::Action
  def initialize
    @skill_id = 1
    @condition_type = 0
    @condition_param1 = 0
    @condition_param2 = 0
    @rating = 5
  end
  attr_accessor :skill_id
  attr_accessor :condition_type
  attr_accessor :condition_param1
  attr_accessor :condition_param2
  attr_accessor :rating
end
