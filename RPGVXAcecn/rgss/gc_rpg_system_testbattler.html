RPG::System::TestBattler

战斗测试的战斗者数据类。

超类

  • Object

调用者

  • RPG::System

属性

actor_id

主角 ID 编号。

level

主角等级。

equips

角色装备，与之相对应的数组中对应的标示如下:

  • 0: 武器
  • 1: 盾牌
  • 2: 头盔
  • 3: 铠甲
  • 4: 饰品

定义

class RPG::System::TestBattler
  def initialize
    @actor_id = 1
    @level = 1
    @equips = [0,0,0,0,0]
  end
  attr_accessor :actor_id
  attr_accessor :level
  attr_accessor :equips
end
