RPG::BaseItem::Feature

特性的数据类。 

超类

  • Object

调用者

  • RPG::BaseItem

属性

code

特性编号。

data_id

特性类型编号（比如攻击附加属性、攻击附加状态）

value

该条特性的具体数值。

定义

class RPG::BaseItem::Feature
  def initialize(code = 0, data_id = 0, value = 0)
    @code = code
    @data_id = data_id
    @value = value
  end
  attr_accessor :code
  attr_accessor :data_id
  attr_accessor :value
end
