RPG::AudioFile

背景音乐、背景音效、音乐、音效的超类。

超类

  • Object

属性

name

音频档名称。

volume

音量（0～100）。背景音乐和音乐的默认值是 100 、背景音效和音效的默认值是 80。

pitch

音调（50～150）。默认值是 100。

定义

class RPG::AudioFile
  def initialize(name = '', volume = 100, pitch = 100)
    @name = name
    @volume = volume
    @pitch = pitch
  end
  attr_accessor :name
  attr_accessor :volume
  attr_accessor :pitch
end
