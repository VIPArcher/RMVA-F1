Tone

色调的类。每个成分使用浮点小数值（Float）。

超类

  • Object

类方法

Tone.new(red, green, blue[, gray])
Tone.new (RGSS3)

生成色调对象。gray 值省略时使用 0。

若没有指定参数，则取默认值为(0, 0, 0, 0)。 (RGSS3)

方法

set(red, green, blue[, gray])
set(tone) (RGSS3)

一次设置所有属性。

第二种方法可以从参数色调类中复制所有属性。 (RGSS3)

属性

red

红色值（-255～255）。超出范围的数值会自动修正。

green

绿色值（-255～255）。超出范围的数值会自动修正。

blue

蓝色值（-255～255）。超出范围的数值会自动修正。

gray

灰度滤镜强度值（0～255）。超出范围的数值会自动修正。

当此值非 0 时，处理时间会比单独使用色调平衡调整来的长。
