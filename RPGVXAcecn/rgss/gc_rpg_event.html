RPG::Event

地图事件块的数据类.

超类

  • Object

调用者

  • RPG::Map

属性

id

地图事件块编号。

name

地图事件块名称。

x

地图事件块在地图上的 X 坐标。

y

地图事件块在地图上的 Y 坐标。

pages

地图事件块事件页， RPG::Event::Page 的数组。

内部类

  • RPG::Event::Page

定义

class RPG::Event
  def initialize(x, y)
    @id = 0
    @name = ''
    @x = x
    @y = y
    @pages = [RPG::Event::Page.new]
  end
  attr_accessor :id
  attr_accessor :name
  attr_accessor :x
  attr_accessor :y
  attr_accessor :pages
end
