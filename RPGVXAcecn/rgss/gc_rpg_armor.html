RPG::Armor

护甲的数据类。

超类

  • RPG::EquipItem

属性

atype_id

护甲类型编号。

方法

performance

护甲评价，通过装备更换界面调用。

返回物理防御、魔法防御以及特性加成效果之和。

定义

class RPG::Armor < RPG::EquipItem
  def initialize
    super
    @atype_id = 0
    @etype_id = 1
    @features.push(RPG::BaseItem::Feature.new(22, 1, 0))
  end
  def performance
    params[3] + params[5] + params.inject(0) {|r, v| r += v }
  end
  attr_accessor :atype_id
end
